Компоненты

Компоненты - это дальнейшее развитие концепции локализации окружения. Можно сказать что это обертка над директивой с наибольшим используемыми настройками по умолчанию. Это особый вид директивы , использующий более простую конфигурацию, подходящую для структуры приложения, основанной на компонентах.

Создание и настройка компонента

Компоненты можно зарегистрировать с помощью .component() метода модуля AngularJS (возвращаемого функцией angular.module()). Метод принимает два аргумента:

  • Имя компонента (в виде строки).
  • Объект конфигурации компонента. (Обратите внимание, что, в отличие от метода .directive(), этот метод не принимает фабричные функции).
