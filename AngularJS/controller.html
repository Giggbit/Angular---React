Контроллеры AngularJS

Лексикографическое округление (scope) - пространство (кода) локализующего область видимости своих объектов ограничивается лексонрафическим околом.
Контроллеры в AngularJS - инструменты для создания отдельных лексографических округлений в одном документе (приложению).

Для создания контроллера необходимо:

  • Дать название (имя) приложения (ng-app="mainApp")
  • Инициализировать приложение (модуль) - в скриптовой части ввыполнить инструкцию ng-controller="nameOfComtroller" HTML-блок, для которого создается отдельный лексографичный округ.
  • Инициализировать контроллер по инмени, что ему было задано mainApp.controller(...) (см. controller.js)
