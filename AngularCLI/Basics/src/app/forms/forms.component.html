Формы Angular. Реактивные формы

Формы (у розумінні html-форми) - это форма обратной связи, средства для передачи данных от пользователя к серверу. Программная задача: повлиять на автоматический процесс отправки формы или полностью ее заменить.

Класический подход - создание html-елментов (input-ов), перехват действия "submit" или замена кнопки на другой тип.

Реактивные формы - соеденение элементов формы с кодом с помощью контролеров, влияние на которые происходит на контролерах, и наоборот. При анализе формы проверяются только данные в контролерах, без необходимости поиска элементов html. Поскольку в большинстве случаев в форме есть несколько элементов, контролеры позволяют группировку для упрощения досупа.
